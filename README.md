# Biomech Retro Horror

A 1990s-inspired retro video game project with oppressive biomechanical horror aesthetics, built for a GitHub + Codex workflow.

## Objective
Create a playable vertical slice first:
- 1 finished level / chamber
- 1 player character
- 2 enemy types
- 1 miniboss
- 1 dialogue / lore system
- 1 title screen
- 1 death / restart loop
- 1 coherent visual biome

## Recommended Stack
- Engine: Phaser 3
- Language: JavaScript or TypeScript
- Repo host: GitHub
- Implementation agent: Codex
- Planning / art direction: GPT-5.4 Thinking

## Suggested Repo Structure

/assets
  /sprites
  /backgrounds
  /ui
  /audio
/docs
/prompts
/src
  /scenes
  /entities
  /systems
  /ui
  /data

## Workflow
1. Read `AGENTS.md`.
2. Read all files in `/docs`.
3. Implement the current milestone only.
4. Keep changes small and modular.
5. Open PR-sized changes, not giant rewrites.
